\doxysubsection{Detailed Description}
This class allows for constructing \mbox{\hyperlink{classFEDD_1_1AssembleFE}{Assemble\+FE}} objects. 


\begin{DoxyTemplParams}{Template Parameters}
{\em SC} & The scalar type. So far, this is always double, but having it as a template parameter would allow flexibily, e.\+g., for using complex instead \\
\hline
{\em LO} & The local ordinal type. The is the index type for local indices \\
\hline
{\em GO} & The global ordinal type. The is the index type for global indices (this should actually by removed since the class should operate only on element level) \\
\hline
{\em NO} & The Kokkos Node type. This would allow for performance portibility when using Kokkos. Currently, this is not used.\\
\hline
\end{DoxyTemplParams}
This class provides a function for constructing objects of classes derived from \mbox{\hyperlink{classFEDD_1_1AssembleFE}{Assemble\+FE}}. In particular, an \mbox{\hyperlink{classFEDD_1_1AssembleFE}{Assemble\+FE}} object is constructed as follows\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{AssembleFE assmeblyFe = \mbox{\hyperlink{classFEDD_1_1AssembleFEFactory_a3fb8f9832f428255d87fae4a5bdd04b1}{AssembleFEFactory<>::build}}(\textcolor{stringliteral}{"problemType"},flag,nodesRefConfig,params);}
\end{DoxyCode}
 If a new element assembly is implemented, it has to be added to the \mbox{\hyperlink{classFEDD_1_1AssembleFEFactory_a3fb8f9832f428255d87fae4a5bdd04b1}{build()}} function. Only then, this element assembly can be executed within the F\+E\+D\+D\+Lib.
