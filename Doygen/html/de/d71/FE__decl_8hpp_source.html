383  // Shape-Derivatives fuer die Geschwindigkeit; die Hauptvariable ist \delta d und Testfunktion ist v:
384  // Spannung 1: DK1 = - \mu \int [ \grad(u) \grad(\delta d) + [\grad(\delta d)]^T [\grad(u)]^T ] : \grad(v) d\Omega
385  // Spannung 2: DK2 = \int [ \sigma \eta ] : \grad(v) d\Omega
386  // mit \sigma = \mu ( \grad(u) + [\grad(u)]^T ) - pI und \eta = I (\nabla \cdot \delta d) - [\grad(\delta d)]^T.
387  // Konvektion: DN = \rho \int (u - w)^T \eta [\grad(u)]^T v d\Omega
388  // Impliziter Anteil der Konvektion: DW = - \rho \frac{1}{k} \int (\delta d \cdot \nabla) u v d\Omega
389  // aehnlich zur Newton-Matrix W aus Navier-Stokes, wobei k = dt = Zeitschrittweite
390  // Zusaetzlicher Term wg. non-conservatie: DP = - \rho \int ( \grad(w) : \eta ) u \cdot v d\Omega
391  // Massematrix und impliziter Anteil wg. non-conservative:
392  // DM = \rho \frac{1.5}{k} \int (\nabla \cdot \delta d) u \cdot v d\Omega (Massematrix aus BDF2)
393  // - \rho \frac{1}{k} \int (\nabla \cdot \delta d) u \cdot v d\Omega
394  // = \rho \frac{0.5}{k} \int (\nabla \cdot \delta d) u \cdot v d\Omega
395  // ACHTUNG: Der erste Koeffizient \frac{1.5}{k} ist durch den fuehrenden Koeffizienten
396  // bei der BDF2-Zeitintegration nach Normierung gegeben (bei BDF2 also 3/2 = 1.5).
